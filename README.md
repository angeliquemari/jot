# jot
Collaborative travel notes editor

#### Requirements

- node.js v10.16.1
- mongoDB v4.2.0

#### Installation

To run locally after cloning the repo, follow the steps below:

Install dependencies, compile client app, run server:
```
npm install
npm run build
npm run start
```
Open browser window with URL: `localhost:3000`
